# SECURITY.md

## Security Policy

### Email Credentials
- Use Gmail app passwords or environment variables.
- Do **not** commit real passwords.

### Data Handling
- Validate all user inputs.
- Sanitize CSV entries to prevent injection.
- Backup critical CSV files regularly.

### Responsible Disclosure
- Report vulnerabilities to Shreeprasandh K.
- Include steps to reproduce the issue and possible mitigation.
